## C++ V1

The key is to identify how to convert this circle-based problem into a linear problem, like LC0198.

We split two cases:

- We don't consider the first house, so only [1:] is considered
- We don't consider the last, so [:-1]

But why split this way? Because we just need to break the tie. 
Notice that, when doing the first branch, not considering the first doesn't mean we must choose the last. So does the second possibility.
We might found a solution that avoids both first and second.

Essentially, there will only be 3 possibilities

| rob first? | rob last? |
| --- | --- |
| n | n |
| y | n |
| n | y |

And we could absorb the first possibility into any of the other two.

```C++
class Solution {
public:
  int rob(vector<int> &nums) {
    if (nums.empty())
      return 0;
    if (nums.size() == 1)
      return nums[0];
    if (nums.size() == 2)
      return std::max(nums[0], nums[1]);
    return std::max(robRange(nums, 0, nums.size() - 2),
                    robRange(nums, 1, nums.size() - 1));
  }
  int robRange(vector<int> &nums, int start, int end) {
    int prev_prev = nums[start];
    int prev = std::max(nums[start], nums[start + 1]);
    int curr = prev;
    for (int i = start + 2; i <= end; i++) {
      curr = std::max(prev_prev + nums[i], prev);
      prev_prev = prev;
      prev = curr;
    }
    return curr;
  }
};
```

Time Complexity: $O(n)$.

Space Complexity: $O(1)$.

## C++ V2

This is another interpretation. Not as concise as the first.

```C++
class Solution {
public:
  int rob(vector<int> &nums) {
    if (nums.empty())
      return 0;
    if (nums.size() == 1)
      return nums[0];
    if (nums.size() == 2)
      return std::max(nums[0], nums[1]);
    if (nums.size() == 3)
      return std::max(std::max(nums[0], nums[1]), nums[2]);
    return std::max(nums[0] + robRange(nums, 2, nums.size() - 2),
                    robRange(nums, 1, nums.size() - 1));
  }
  int robRange(vector<int> &nums, int start, int end) {
    if (start == end)
      return nums[start];
    int prev_prev = nums[start];
    int prev = std::max(nums[start], nums[start + 1]);
    int curr = prev;
    for (int i = start + 2; i <= end; i++) {
      curr = std::max(prev_prev + nums[i], prev);
      prev_prev = prev;
      prev = curr;
    }
    return curr;
  }
};
```

Time Complexity: $O(n)$.

Space Complexity: $O(1)$.
